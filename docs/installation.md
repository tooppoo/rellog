# Installation

## curl installe(macOS / Linux)

```sh
curl -fsSL https://raw.githubusercontent.com/tooppoo/rellog/refs/heads/main/install.sh | sh
```

Review-first alternative

```sh
curl -fsSLO https://raw.githubusercontent.com/tooppoo/rellog/refs/heads/main/install.sh
sh ./install.sh --help
sh ./install.sh
```

## download archive(macOS / Linux / Windows)

Download the archive for your architecture from [GitHub Releases](https://github.com/tooppoo/rellog/releases) and extract it.
